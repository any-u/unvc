# unvc

[![NPM version](https://img.shields.io/npm/v/unvc?color=a1b858&label=)](https://www.npmjs.com/package/unvc)

A universal solution for check page visibility.

## Usage

```bash
npm i unconfig
```

```js
import { vc } from 'unvc'

vc(
  // onHide
  () => {

  },
  // onShow
  () => {

  }
)
```
